### Gráficos ###
A parte gráfica está bem simples, no geral foram usados os modelos de primitivas da propria unity. Apenas a moeda utiliza um modelo gratuito do site free3d.com. Como não era uma exigencia do teste priorizei as funcionalidades.

### História ###

O jogo possui em enredo simples, onde um personagem foi envenenado e precisa passar por um corredor antes que morra para conseguir o antidoto.

###Elementos###
Player  -> Cubo preto controlavel.
NPC -> Cubo preto no inicio do cenário.
Cura -> Cubos verde espalhados pelo cenário que ao serem destruidos recuperam vida.
Upgrades -> Esferas coloridas que quando coletadas liberam as habilidades de tiro para o jogador.
Moedas -> Modelos espalhados no cénario que podem ser coletados pelo jogador. 

### Controles ###

Movimentação -> setas do teclado
Pulo -> Barra de espaço
Tiro vermelho -> Q
Tiro verde -> W
Tiro azil -> E

Alguns inputs padrões da Unity foram modificados, para que a ação de atirar não entrasse em conflito com a de andar por exemplo. 

### Build ###
Todos os testes e a build final do jogo foram feitos no sistema Windows, porém o jogo deve funcionar perfeitamente se exportado para outros sistemas.
